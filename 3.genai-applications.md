## 0 — What is OpenRouter?

OpenRouter is a single API gateway that gives you access to many AI models
(Gemini, DeepSeek, Qwen, MiniMax, and more) through one unified API.

```
Your Code  →  OpenRouter API  →  google/gemini-3.5-flash
                              →  deepseek/deepseek-v4-pro
                              →  qwen/qwen3.7-max
                              →  minimax/minimax-m3
```

One key. One SDK. Multiple models. You can switch models by changing one line.

## 1 — Setup

### 1.1 — Install dependencies
```bash
pip install openai python-dotenv
```
> OpenRouter uses the same SDK as OpenAI — only the `base_url` changes.

### 1.2 — Create your `.env` file
```bash
copy .env.example .env        # Windows
cp .env.example .env          # Mac/Linux
```
Open `.env` and fill in your key:
```
OPENROUTER_API_KEY=your-actual-key-here
```

### 1.3 — Verify `.gitignore` has `.env`
```bash
# Check this exists in .gitignore before every push
cat .gitignore | grep .env
```

### 1.4 — Test your key (smoke test)
```bash
python applications/01_basic_ask.py
```
If you see a response about Kubernetes namespaces, your key is working.

## 3 — The OpenRouter Code Pattern

Every application in this repo follows this same pattern:

```python
import os
from openai import OpenAI
from dotenv import load_dotenv

load_dotenv()

client = OpenAI(
    api_key=os.getenv("OPENROUTER_API_KEY"),
    base_url="https://openrouter.ai/api/v1"   # ← only difference from OpenAI
)

response = client.chat.completions.create(
    model="google/gemini-3.5-flash",           # ← swap model here
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user",   "content": "Your question here"}
    ]
)

print(response.choices[0].message.content)
```

## 6 — API Key Safety Rules

1. Never paste your key directly in any `.py` file
2. Always load from `.env` using `python-dotenv`
3. `.env` is in `.gitignore` — confirm before every `git push`
4. Never push to a public GitHub/GitLab repository
5. If the key is ever accidentally exposed, report to your session lead immediately
